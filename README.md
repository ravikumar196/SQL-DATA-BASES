This project focuses on analyzing hospital data using SQL.
The dataset includes information about hospitals, departments, patients, doctors, medical expenses, and admission/discharge dates.

Below are the key questions addressed in this project:

Questions

Total Number of Patients
Find the total number of patients across all hospitals.

Average Number of Doctors per Hospital
Retrieve the average count of doctors available in each hospital.

Top 3 Hospitals with Highest Number of Patients
Find the top three hospitals with the highest patient count.

Hospital with Maximum Medical Expenses
Identify the hospital that recorded the highest medical expenses.

Daily Average Medical Expenses per Hospital
Calculate the average medical expense per day for each hospital.

Longest Hospital Stay
Determine which patient had the longest hospital stay.

Total Patients Treated per City
Count the total number of patients treated in each city.

Average Length of Stay per Department
Calculate the average number of days patients spend in each department.

Department with the Least Number of Patients
Identify the department with the lowest patient count.

Monthly Medical Expenses Report
Group the data by month and calculate the total medical expenses for each month.

This list of questions serves as a roadmap for hospital data analysis, helping to track patient flow, hospital performance, and medical expenses trends.
